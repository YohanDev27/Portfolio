Conception de site web

Aujourd’hui un grand nombre d’entreprises ou de particuliers confectionnent leurs propres sites web. Il permet à son créateur de mettre en avant ses services, ses produits ou tout simplement se présenter. La conception d’un site web peut se traduire par le fait de créer de toute pièce une interface Web par le biais de trois technologies, HTML, CSS et Javascript. Chacun ici a son rôle à jouer dans la création d’un site web :

  • L’HTML pour Hypertyext Markup Language est un langage de balisage qui permet de rendre l’affichage d’une page web réalisable
  • CSS pour Cascading Style Sheets met en forme les pages web en changeant par exemple la couleur du texte ou en modifiant la position d’un bouton
  • JS pour Javascript donne la possibilité à une page web d’avoir une logique

La notion de « branche » est importante avec Git. Une branche peut être traduite par une copie du projet pour développer et tester de nouvelles fonctionnalités sans impacter le projet de base. Cette branche peut ensuite être « merge » (fusionné) avec la branche principale « Main » du projet pour y apporter ces modifications.

Cette compétence est très souvent utilisée dans le monde professionnel. Elle permet à des entreprises de se démarquer de leurs concurrents en proposant des sites web adaptés à leurs besoins. En plus de ça, la conception de site web peut répondre à de nombreux besoins pour des solutions utilisées en interne dans les entreprises. Il est important de noter qu’il existe aujourd’hui des outils permettant de faciliter la création de site web basique comme WordPress, Wix ou encore Joomla. Ces outils ne sont néanmoins pas conseillés pour la conception de site web complexe.

Je travaille aujourd’hui à Capgemini en collaboration avec Air-France en tant qu’alternant et je constate que sur mon périmètre, une multitude de sites web voient le jour dans le but d’améliorer les conditions de travail des techniciens Air-France.

Mise en oeuvre

Lors de mon premier semestre à IN’TECH, j’ai eu l’opportunité de découvrir les langages HTML, CSS et Javascript pour le développement de mon tout premier projet, KeepClean. KeepClean est un jeu vidéo en 2D qui a pour but de sensibiliser les joueurs sur le tri des déchets.


Exemple du site web KeepClean

Pour le développement du projet je devais créer la collision de la poubelle avec les déchets qui tombaient. Pour ce faire je devais d’abord comprendre comment l’axe X de la poubelle et l’axe Y des déchets fonctionnaient pour commencer à développer la collision. Une fois compris j’ai alors codé en Javascript la collision des objets pour ensuite incrémenter le score du joueur.

Je suis aussi intervenu sur la création du choix des niveaux avec une carte interactive. Cette partie demandait de bien comprendre comment l’HTML et le JavaScript fonctionnaient ensemble. Je devais en effet créer une carte en SVG pour la rendre interactive. L’ajout de Javascript a permis de rendre le clic possible sur la carte avec un chargement d’un niveau.


Exemple de la carte interactive

Grâce à la rapidité de la confection du site web, le jeu a pu voir le jour et a été présenté pendant la fête du Jeu à Dax. En plus de ça nous avons eu la chance d’être interviewé et de bénéficier d’un article de presse.

Maitrise

Pour la conception de site web j’estime être confirmé.

Pour moi cette compétence est très importante pour ce qui est du métier de développeur. De nombreuses entreprises sont demandeuses de cette compétence pour les accompagner dans la création de sites web personnalisés qui peuvent être souvent complexes. C’est pour cela que l’ingénieur se doit d’être capable de bien maitriser la conception d’un site web.

Cette compétence nous apprend la base du développement par le biais de ces langages qui sont HTML, CSS et Javascript. J’ai pu apprendre rapidement comment confectionner un site web puisque j’ai eu l’opportunité de pratiquer les trois langages sur tous mes projets jusqu’à maintenant. Néanmoins, il me reste une certaine marge de progression dans le domaine du design et de l’ergonomie d’un site web. L’utilisation de cette compétence peut varier suivant la demande. Par exemple la création d’un site vitrine va souvent demander l’utilisation d’outil simple comme WordPress mais pour la création d’un site web complexe il sera préférable de développer via HTML, CSS et Javascript.

Avec l’expérience que j’ai pu acquérir, je conseille à autrui de commencer par apprendre les bases de l’HTML, CSS et Javascript. Elles sont pour moi nécessaire pour confectionner un site web. Pour ce faire de nombreux sites sont à disposition pour apprendre facilement. On peut citer W3Schools pour ses tutoriels simples et divers ou encore Open Class Rooms qui lui propose des formations en français riches en contenu.

Évolution compétence

Cette compétence me suit au quotidien dans mon travail et je souhaite l’améliorer pour ce qui est de l’aspect design et ergonomique. En effet il m’est souvent demandé de créer des pages web ergonomique pour les utilisateurs. Pour ce faire je compte m’inscrire à la formation de « M2iFormation » portant sur l’ergonomie des sites web.

Lien de la formation : M2iformation

Cette formation va me permettre :

  • D’identifier les enjeux et les critères de l'ergonomie d'interface pour le Web
  • D’appliquer l'ergonomie adapté à chaque utilisateur
  • D’évaluer la qualité de vos interfaces.
